Let's go through some types

  • Deciduous Trees
    • Also called hardwoods, these tree types typically shed their leaves
    • Deciduous trees have different-shaped leaves, depending on tree species;
    • Examples: oak (Quercus spp.), elm (Ulmus spp.), maple (Acer spp.), willow (Salix spp.)
  • Coniferous Trees
    • Most of these are evergreens.
    • Coniferous trees have leaves that are needle-shaped
